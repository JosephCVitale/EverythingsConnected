Guide to Everything's Connected

What it Does

This app allows you to find all the artists that someone has collaborated with on Spotify. It uses the Spotify API (the part of their database the public can see), to scan through all of an artist's songs and pull out every artist that was featured on them. You then have to ability to do the same thing to any those artists, and create a big network showing how a lot of artists have some connection to another. It's kinda like the famous Kevin Bacon Number thing. You have to log in to spotify use the app. For more information on why check here. If you want to learn more about me or app itself you can look here.

Controls

There are a few ways to use this app. After you've logged in you type in any artist's name and their circle will appear, then it will do a search for all of their collaborators. Depending on how prolific the artist is, this might take a few seconds.

One at a Time Mode

This will only show the artists within 1 degree of separation from the artist you type, then you can click on any of those artists to run the same search on them to create another layer. This is my recommended mode since it runs faster and it's easier to see the information.

Full Search Mode

This will keep doing a recursive search on every artist found up to a certain degrees of separation. You can set the degree limit with the slider right below it. This can be cool to see the shear number of artists in the network, but it can take a while to load them all, and it's hard to see what's going on when there's a couple thousand artists shown. I only recommend going to two degrees unless you know what you're doing or you're willing to wait a while.

Show Images

Pretty simple. Shows images of the artist in their node. It looks better, but at a high volume it can cause a lot off lag. When this is off each node will be a unique color based on it's degrees of separation.

Base Circle Size on Popularity

Spotify assigns every artist a popularity score between 0 and 100. With this setting on, the radius of each circle will be equal to that popularity. This can help you pick out the more famous artists. If you want equal representation then you can turn this off and each circle will be the same size.

Song connecting the artists

If you click on the line between two artists, it will show the name of the song they collaborated on in the sidebar. It will also play a 30 second sample of that song. It's quite possible they collaborated on more than one song, but this just shows the first instance that is found.

Delete

The trash can icon. Will delete the entire network and start over.

Mute

Mutes the audio when you click on a link.

Games You Can Play

I have fun putting it on the one at a time mode, and trying to find a way to connect two disparate artists. You'd be amazed at some of the connections you can make with only a few degrees.

Theoretically with most artists you can go functionally forever expanding nodes until you reach a point where you have virtually every artist represented, but you're browser would probably crash before that happened. I want to put a challenge out there to find the artist with the largest network you can with no more connections to be found. This means expanding every artist until there's no more.
